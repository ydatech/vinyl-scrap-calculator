# Vinyl Scrap Calculator

A simple calculator used by technicians to estimate vinyl usage when cuttin graphics.


## Features 
- Roll width input
- Graphic dimension input
- Calculate pieces per row
- Area calculation
- Mobile responsive


## CI/CD Pipeline
The project uses **GitHub Actions**
Pipeline stages:
1. Lint HTML, CSS, and JS
2. Deploy to GitHub Pages

Pipeline triggers automatically when pushing to the `main` branch.

## Manual Deployment
You can trigger deploy manually:
1. Go to **GitHub**
2. Open **Actions**
3. Select **Deploy Vinyl Calculator**
4. Click **Run wokrflow**
